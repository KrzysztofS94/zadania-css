Podaj definicję modelu pudełkowego

\ W CSS każdy element HTML można przedstawić jako model pudełkowy (zwany również modelem blokowym).
Z wykorzystaniem modelu pudełkowego, możemy sterować wyglądem i układem strony. Na model pudełkowy składa się zawartość/treść,
margines wewnętrzny, obramowanie oraz margines zewnętrzny.
